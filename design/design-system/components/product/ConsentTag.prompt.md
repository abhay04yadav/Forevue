DPDP privacy marker for minors — Forevue never surfaces a minor's data without it.

```jsx
<ConsentTag state="required" />   {/* amber lock — "Minor — consent required" */}
<ConsentTag state="granted" />    {/* green check — "Consent on file" */}
```

Default to `required` for any record flagged as a minor. This is a governance signal, not decoration — keep the copy plain and present it near the student's identity.
